Prompt工程介绍

Prompt = 给AI一个指令/列表，使它产生预期结果

5 / 20

Prompt写作格式

1. 身份设定

"你是一个专业统计分析师"

2. 对话规则设定

用"system"或"user"设定对话规则

3. 结果格式指定

"以markdown格式输出" "必须使用python代码块"

4. 示例提供

提供既有的input-output对

Prompt工程经典模式

Zero-shot

直接提问

One-shot

给一个示例

Few-shot

给出3~5个示例

Chain-of-thought

强调分步理解和解题

ReAct

给出"思考+操作"链

后端服务搭建基础

后端（Backend）是软件系统中负责处理逻辑、数据、计算与接口的部分

8 / 20

后端的主要职责

1. 接收请求

处理前端发来的各类请求

2. 数据处理

涉及数据库、AI模型等操作

3. 业务逻辑

实现核心功能和算法

4. 返回结果

将处理结果返回给前端

后端常见名词解释

名词 解释
API 应用程序接口，前后端交流的"约定通道"
路由 (Route) 指定网址对应的后端处理函数
请求 (Request) 用户发给服务器的信息
响应 (Response) 服务器返回给用户的信息

后端代码执行流程

3. 请求LLM获取回答

4. 保存对话历史
